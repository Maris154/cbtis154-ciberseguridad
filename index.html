Perfil de Egreso

Como Técnico en Ciberseguridad serás capaz de proteger sistemas informáticos contra amenazas cibernéticas, empleando herramientas y procedimientos que mantienen la información segura en el mundo digital.

Competencias que desarrollarás:

  • Desarrollar algoritmos para custodiar la seguridad de la información
  • Instalar sistemas operativos para asegurar la información
  • Mitigar riesgos de seguridad en sistemas informáticos
  • Monitorear el tráfico de redes para detectar vulnerabilidades
  • Auxiliar en la preservación de evidencias de ataques informáticos

Ejemplo para empezar: Piensa en la tienda de tu colonia. Qué información manejan? Qué pasaría si alguien hackea su sistema?

Van a crear un manual de seguridad completo para un tipo de negocio local. El objetivo es que sea tan bueno que un dueño de negocio real pudiera usarlo.

Giros de negocio disponibles:

Restaurante

Ventaja: Puedes ser creativo imaginando escenarios.
Reto: Debes investigar cómo funcionan estos negocios.
Ejemplo para empezar: Te imaginas crear tu propio verificador de archivos como los que usan los antivirus? Eso harás aquí.

Van a desarrollar un conjunto de scripts y herramientas de seguridad que puedan aplicarse a cualquier organización. Es como crear un kit de primeros auxilios digitales.

Ventaja: Mayor libertad técnica para crear herramientas.
Reto: Debes demostrar que tus herramientas funcionan.
Roles del equipo (5-9 integrantes)

Coordina al equipo, organiza las entregas, presenta avances, resuelve conflictos.

Perfil: Alguien organizado que sepa comunicarse bien.

  • Crear y mantener el calendario de entregas
  • Asignar tareas y dar seguimiento
  • Ser el contacto principal con el docente
  • Presentar los avances en clase

Identifica activos, amenazas, vulnerabilidades y riesgos. Busca scripts útiles.

Perfil: Alguien curioso que le guste investigar.

  • Crear el inventario de activos
  • Identificar amenazas y vulnerabilidades
  • Calcular niveles de riesgo

Diseña los algoritmos, crea diagramas de flujo, define cómo funcionará el sistema.

Perfil: Alguien que piense de forma lógica y ordenada.

  • Diseñar diagramas de flujo
  • Escribir pseudocódigo
  • Definir la estructura del sistema

Escribe pseudocódigo, adapta scripts existentes, programa en Python.

Perfil: Alguien que le guste programar.

  • Adaptar scripts al proyecto
  • Escribir código Python funcional
  • Implementar los diseños

Diseña e implementa soluciones de cifrado, hash y verificación de integridad.

Perfil: Alguien bueno con matemáticas y detalles.

  • Implementar funciones de hash (SHA-256)
  • Configurar cifrado AES correctamente
  • Verificar integridad de archivos

Prueba todo, encuentra errores, verifica que funcione correctamente.

Perfil: Alguien meticuloso que le gusta romper cosas.

  • Probar cada script con diferentes datos
  • Documentar errores encontrados
  • Validar que el sistema funcione

Redacta manuales, políticas, reportes técnicos y presentaciones.

Perfil: Alguien que escriba bien y sea ordenado.

  • Escribir el README de cada entrega
  • Crear manuales de usuario
  • Preparar las presentaciones

Elabora y codifica algoritmos para la solución de problemas de seguridad informática utilizando diagramas de flujo y pseudocódigo.

Competencias a desarrollar
  • Analiza problemas de seguridad informática
  • Diseña algoritmos con diagramas de flujo
  • Realiza pseudocódigo de algoritmos
  • Codifica algoritmos en lenguaje de programación
Contenido del curso

Desarrolla y ejecuta scripts en Python para automatizar tareas de seguridad, adaptando código existente y documentando soluciones.

Competencias a desarrollar
  • Identifica scripts de seguridad existentes
  • Adapta scripts para solución de problemas
  • Verifica funcionamiento en ambiente de pruebas
  • Genera evidencia digital documentada
